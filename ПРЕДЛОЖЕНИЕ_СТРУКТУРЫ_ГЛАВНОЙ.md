### Что есть сейчас:
1. **Общая статистика** (4 карточки):

   - Поставщики
   - Подкатегории

2. **Графики**:

   - Последние товары (10)
   - Товары на проверке (5)

5. **Быстрые действия** (3 кнопки)

### Проблемы текущей структуры:
- ❌ Нет иерархии: Категория → Поставщик → Подкатегория
- ❌ Нет детальной информации по категориям

- ❌ Нет информации о валидации по категориям/поставщикам
- ❌ Нет информации о загруженных файлах
- ❌ Нет информации о наполнении подкатегорий

## 🎨 ВИЗУАЛЬНАЯ СТРУКТУРА

### Вариант 1: Аккордеон (рекомендуется)
- Категории раскрываются/сворачиваются
- Внутри категории - поставщики (раскрываются/сворачиваются)
- Внутри поставщика - подкатегории (раскрываются/сворачиваются)
- Компактно, не перегружает страницу

### Вариант 2: Дерево (tree view)
- Полная иерархия видна сразу
- Можно раскрывать/сворачивать узлы
- Более наглядно, но может быть перегружено

### Вариант 3: Табы
- Вкладка "По категориям"
- Вкладка "По поставщикам"
- Вкладка "По подкатегориям"
- Менее иерархично, но удобно для фильтрации

## 🎯 РЕКОМЕНДУЕМАЯ СТРУКТУРА (ИТОГ)

### Верхняя часть (всегда видна):
1. Ключевые метрики (4 карточки)
2. Статусы валидации (4 карточки)
3. Средняя оценка верификации

### Основная часть (иерархия):
1. **Категории** (аккордеон)
   - Статистика категории
   - **Поставщики** (аккордеон)
     - Статистика поставщика
     - **Подкатегории** (список с деталями)
       - Статистика подкатегории
       - Наполнение подкатегории
     - Файлы для валидации
   - График по категории (опционально)

### Нижняя часть:
1. Последние импорты
2. Требуют внимания
3. График динамики (опционально)

## 💡 ПРЕИМУЩЕСТВА ПРЕДЛАГАЕМОЙ СТРУКТУРЫ

1. ✅ **Иерархия видна сразу** - Категория → Поставщик → Подкатегория
2. ✅ **Детальная информация** - по каждому уровню отдельно
3. ✅ **Валидация на виду** - статусы и оценки видны сразу
4. ✅ **Файлы для валидации** - история импортов по поставщикам
5. ✅ **Наполнение подкатегорий** - видно, сколько товаров заполнено
6. ✅ **Компактность** - аккордеон не перегружает страницу
7. ✅ **Навигация** - легко перейти к нужному разделу
