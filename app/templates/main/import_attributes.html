Импорт атрибутов

Загрузка атрибутов из файла

Назад
Загрузка файла
Поддерживаемые форматы: Excel (.xlsx, .xls), CSV, JSON
Формат файла

Обязательные поля:

  • code - Код атрибута (уникальный)
  • name - Название атрибута
  • type - Тип (text, number, date, boolean, url, image, select)

Опциональные поля:

  • description - Описание
  • unit - Единица измерения (кг, м, шт и т.д.)
  • is_unique - Уникальный атрибут (true/false)

Пример CSV:

code,name,type,unit,description
weight,Вес,number,кг,Вес товара
length,Длина,number,м,Длина товара
color,Цвет,text,,Цвет товара
